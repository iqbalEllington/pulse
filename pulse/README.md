## NextEcommerce

---

Reealestate website with React, Redux, Redux-Thunk, Next JS, React Hook API

### Technology
1. React
1. Redux
1. Redux-Thunk
1. Next JS

node version 14.17.5
## How to run

1. Clone Project -
```bash
git clone 
```
1. Go to EcommerceAdvanceFrontend folder
```bash
cd EcommerceAdvanceFrontend
```
1. Install npm
```bash
npm i
```
1. Run Project
```bash
npm run dev
```
1. Check Browser. The frontend Started on below address
```bash
http://localhost:3000
```
